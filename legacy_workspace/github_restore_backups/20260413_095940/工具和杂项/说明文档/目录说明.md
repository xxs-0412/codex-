# 目录说明

- `寿命预测软件.bat`：直接启动寿命预测软件界面。
- `磨损数据`：原始磨损数据，文件名已经改成中文试验编号。
- `磨损数据（改）`：修改后的标准化单试验数据，保留 `修改数据汇总.csv` 和各个 `试验*.csv`。
- `结果`：各类对比结果、图片和汇总表。
- `软件主体`：程序本体和界面代码。
- `软件主体\虚拟数据`：旧的虚拟样本数据目录，主要给旧训练脚本使用。
- `软件主体\app_workspace`：预测软件界面的工作区，主要放输入参数文件 `prediction_input.csv`，以及一个备用启动 bat。
- `工具和杂项`：脚本、配置、日志、训练模型、说明文档、临时文件、缓存和编辑器设置。
- `工具和杂项\编辑器设置\.vscode`：VS Code 的项目设置，不影响软件运行。
- `legacy_workspace`：保留旧版本工作区和旧开发备份，目前包含 `previous_version` 与收进去的 `recursive_physics_surrogate`。

## 常用入口

- 启动软件：双击 `寿命预测软件.bat`
- 重新处理并训练：双击 `工具和杂项\脚本与配置\一键重新处理并训练.bat`
- 运行网络结构对比：双击 `工具和杂项\脚本与配置\运行网络结构对比.bat`
